1. Чему равно число 10102 в десятичной системе счисления?

1·2³ + 0·2² + 1·2¹ + 0 = 8 + 2 = 10.

2. Сколько бит минимум нужно, чтобы закодировать один символ алфавита из 256 символов при равномерной длине кода?

Нужно найти минимальное k: 2k ≥ 256 → k = 8.

3. Чему равно выражение (1 ∧ 0) ∨ 1 в алгебре логики? (1 — истина, 0 — ложь)

(1 ∧ 0) = 0; 0 ∨ 1 = 1.

4. Переведите число 2D16 в десятичную систему (D = 13).
