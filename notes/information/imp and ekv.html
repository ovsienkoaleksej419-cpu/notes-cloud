Задание 2: Конструирование таблиц истинности

В задании дан фрагмент таблицы истинности для логического выражения. Нужно определить, какому столбцу таблицы соответствует каждая из переменных (x, y, z, w).

Главный секрет: Не пытайся сопоставить переменные в уме. Напиши код, который выведет все строки, где выражение принимает нужное значение (0 или 1), и сравни результат с КИМом.

1. Логические операции в Python

Операция Символ в ЕГЭ Код в Python
Инверсия (НЕ) ¬ или ¬x not x
Конъюнкция (И) ∧ или ⋅ x and y
Дизъюнкция (ИЛИ) ∨ или + x or y
Импликация (Следование) → x <= y
Эквивалентность (Тождество) ≡ или ↔ x == y

2. Универсальный шаблон решения

Допустим, дано выражение: (x → y) ∧ (y → z) ∧ (z → w), и оно должно быть истинным.

print("x y z w") # Печатаем заголовок для удобства
for x in 0, 1:
    for y in 0, 1:
        for z in 0, 1:
            for w in 0, 1:
                # Записываем формулу:
                f = (x <= y) and (y <= z) and (z <= w)
                
                # Если в условии таблицы стоят 1 (True), проверяем if f:
                # Если в условии таблицы стоят 0 (False), проверяем if not f:
                if f:
                    print(x, y, z, w)

3. Алгоритм сопоставления (без воды)

  1. Запусти код и получи свою "чистую" таблицу.
  2. Посмотри на количество единиц и нулей в столбцах твоей таблицы.
  3. Сравни их со столбцами во фрагменте из задания.
    • Например: если в твоей таблице только в одном столбце (допустим, w) три нуля, а в фрагменте КИМа тоже только в одном столбце три нуля — значит, этот столбец и есть w.
  4. Ищи уникальные строки (где только одна единица или только один ноль).
Важно: Всегда соблюдай приоритет! Самый низкий приоритет у эквивалентности (==) и импликации (<=), поэтому всегда бери их в скобки, чтобы Python не выполнил and раньше времени.
