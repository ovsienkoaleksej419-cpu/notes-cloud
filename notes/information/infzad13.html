Задание 13: IP-адресация и маски

IP-адрес состоит из 32 бит (4 байта по 8 бит). Маска подсети определяет, какая часть адреса относится к адресу сети, а какая — к номеру узла.

Как работает маска?

Маска — это всегда последовательность единиц, а затем нулей. Например:

255.255.255.240 → 11111111.11111111.11111111.11110000

Единицы в маске "замораживают" часть IP-адреса, делая её адресом сети. Нули — это "свободное место" для адресов компьютеров.

1. Ручной способ (Поразрядное И)

Адрес сети получается путём применения операции И (AND) между IP-адресом и маской.

ОбъектДесятичный видДвоичный вид (последний байт)
IP-узел... .15010010110
Маска... .24811111000
Сеть... .14410010000

2. Решение через Python (Библиотека ipaddress)

Это самый надежный способ на ЕГЭ. Он исключает ошибки в переводах систем счисления.

import ipaddress

# Создаем объект сети: "IP-узла/маска"
# strict=False позволяет передать адрес узла вместо адреса сети
net = ipaddress.ip_network('192.168.10.150/255.255.255.248', strict=False)

print(f"Адрес сети: {net.network_address}")
print(f"Всего адресов: {net.num_addresses}")

# Перебор всех адресов в сети (за вычетом адреса сети и широковещательного)
count = 0
for addr in net:
    # Условие: например, в двоичной записи узла больше единиц, чем нулей
    b = f"{addr:b}"
    if b.count('1') > b.count('0'):
        count += 1

print(f"Подходящих адресов: {count}")

Частые вопросы:

  • Сколько узлов в сети? Формула: 2n - 2 (где n — количество нулей в маске).
  • Что такое широковещательный адрес? Это последний адрес в сети (все нули маски заменены на единицы в IP).
  • Prefix length (/24): Это количество единиц в маске. /24 означает 255.255.255.0.
