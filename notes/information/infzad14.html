Задание 14: Позиционные системы счисления

Задание проверяет умение работать с числами в разных системах счисления (СС), находить остатки и переводить выражения.

Два главных правила:

  1. Чтобы найти последнюю цифру числа в СС с основанием N, нужно взять число % N.
  2. Чтобы отбросить последнюю цифру, нужно сделать число // N.

Тип 1: Выражение с переменной X

Задача: Найти значение x, при котором выражение делится на 15. Значения в разных СС.

for x in "0123456789ABCDEFGHIJ": # Все возможные цифры
    # Переводим из СС с основанием 20 и 25: int(строка, основание)
    num1 = int(f"123{x}5", 20)
    num2 = int(f"1{x}234", 25)
    
    val = num1 + num2
    if val % 15 == 0:
        print(val // 15)
        break

Тип 2: Подсчет количества цифр

Задача: Сколько цифр "4" в записи числа $5 \cdot 25^{15} + 4 \cdot 5^{10} - 15$ в 5-ричной системе?

n = 5 * 25**15 + 4 * 5**10 - 15
count_4 = 0

while n > 0:
    if n % 5 == 4: # Если последняя цифра в 5-ричной СС равна 4
        count_4 += 1
    n //= 5 # Убираем последнюю цифру

print(count_4)

Тип 3: Поиск основания системы (p)

Иногда нужно найти само основание p. Просто перебираем его циклом.

for p in range(2, 100):
    try:
        if int("81", p) == int("101", 9):
            print(p)
    except: continue # Если цифра 8 не влезает в СС, пропускаем

Шпаргалка:

  • int(s, p) — перевод из системы p в десятичную.
  • Алфавит для СС выше 10: 0..9, A(10), B(11), C(12), D(13), E(14), F(15)...
  • В Python 3.11+ нет лимита на количество цифр в int, но если число гигантское, используй строки осторожно.
  • Если просят "наибольшее x" — убирай break и бери последнее значение.
