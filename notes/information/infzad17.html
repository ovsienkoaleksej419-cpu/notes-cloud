Задание 17: Работа с массивами и файлами

В этом задании нужно прочитать список чисел из файла, найти количество пар элементов, удовлетворяющих условию, и найти какую-то характеристику (например, максимальную сумму).

Что такое "пара" в 17 задании?

Внимательно читай условие, это критично:

  • Подряд идущие: это элементы a[i] и a[i+1].
  • Произвольные пары: это любые два числа из списка (встречается редко, решается через два вложенных цикла).

1. Этап первый: Чтение файла и поиск константы

Часто условие привязано к какому-то числу из файла (например, "минимальное число, оканчивающееся на 13").

# 1. Читаем все числа в список
data = [int(x) for x in open('17.txt')]

# 2. Ищем опорное число (например, макс. элемент кратный 10)
max_10 = max(x for x in data if x % 10 == 0)

2. Этап второй: Проход по парам

Используем цикл range(len(data) - 1), чтобы не выйти за границы списка.

and a[i+1]] 
ans = []
for i in range(len(data) - 1):
    v1 = data[i]
    v2 = data[i+1]
    
    # Пример условия: ровно одно число из пары делится на 7, 
    # а их сумма меньше нашего max_10
    cond1 = (v1 % 7 == 0) + (v2 % 7 == 0) == 1
    cond2 = (v1 + v2) < max_10
    
    if cond1 and cond2:
        ans.append(v1 + v2)

# В ответе обычно просят количество и максимум
print(len(ans), max(ans))

3. Полезные фишки для условий

Чтобы код был чище, используй логическое сложение (True = 1, False = 0):

  • (x % 3 == 0) + (y % 3 == 0) == 1 — ровно одно из двух.
  • (x % 3 == 0) + (y % 3 == 0) >= 1 — хотя бы одно из двух.
  • x % 10 == 3 — число оканчивается на 3 (для положительных чисел).
  • abs(x) % 10 == 3 — число оканчивается на 3 (обязательно для отрицательных чисел!).

Чек-лист для 17 задания:

  • Отрицательные числа: Если в файле есть минусы, x % 10 может работать не так, как ты ждешь. Всегда пиши abs(x) % 10.
  • Границы цикла: Не забудь len(data) - 1 для пар и len(data) - 2 для троек.
  • Имена переменных: Не называй список list, это зарезервированное слово в Python.
