Задание 15: Программирование логики

В этом задании нужно найти такое A, при котором выражение истинно (равно 1) при любом x (и y, если он есть).

Таблица соответствия операторов:
  • И (Конъюнкция): and
  • ИЛИ (Дизъюнкция): or
  • НЕ (Инверсия): not
  • Следование (Импликация): <= (или not x or y)
  • Тождество: ==

Тип 1: Делимость (ДЕЛ)

Задача: Найти минимальное натуральное А, для которого выражение истинно.

def D(n, m):
    return n % m == 0

for a in range(1, 1000):
    ok = True
    for x in range(1, 1000):
        # Записываем выражение из условия:
        f = (D(x, 2) <= (not D(x, 3))) or (x + a >= 100)
        if not f:
            ok = False
            break
    if ok:
        print(a)
        break # Нашли минимальное - выходим

Тип 2: Поразрядная конъюнкция (&)

Ключевой оператор в Python: &. Важно помнить про приоритет операций (всегда ставь скобки!).

for a in range(0, 1000):
    if all(((x & 29 != 0) <= ((x & 17 == 0) <= (x & a != 0))) for x in range(1000)):
        print(a)
        break

Тип 3: Отрезки

Для отрезков $[P, Q]$ проверяем вхождение числа: P <= x <= Q.

# Допустим P = [10, 20], Q = [25, 35]
p = [i for i in range(10, 21)]
q = [i for i in range(25, 36)]
a = [] # Ищем элементы, входящие в А

for x in range(1, 100):
    # Если при ложном А выражение ложно, значит x должен быть в А
    f = ((x in a) <= (x in p)) or (x in q)
    if not f:
        a.append(x)
print(sorted(a))
Тонкий момент:
  • Функция all() — это быстрый способ проверить условие для всех x.
  • Если ищем минимум — печатаем первый подошедший a и делаем break.
  • Если ищем максимум — печатаем все и смотрим последнее число.
  • Диапазона range(1000) обычно хватает для 99% задач ЕГЭ.
