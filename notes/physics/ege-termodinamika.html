Количество теплоты

Q = cmΔT
Q = λm   (плавление/кристаллизация)
Q = Lm   (парообразование/конденсация)

Первое начало термодинамики

ΔU = Q + A
для газа: A = −pΔV (работа газа при p=const: A = pΔV внешняя)

Знак работы: согласуйте с формулировкой задачи (работа над газом или газа).

Изопроцессы (напоминание)

Изотерма: pV = const. Изохора: Q = νCVΔT. Изобара: Q = νCpΔT.

КПД тепловой машины

η = A/Q_1 = (Q_1 − Q_2)/Q_1
η_Карно = 1 − T_2/T_1
