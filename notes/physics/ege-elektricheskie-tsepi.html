ЕГЭ физика · Закон Ома, последовательное и параллельное соединение

Закон Ома для участка и полной цепи

I = U/R
I = ℰ/(R + r)

ℰ — ЭДС источника, r — его внутреннее сопротивление.

Соединение резисторов

R_посл = R₁ + R₂ + …
1/R_пар = 1/R₁ + 1/R₂ + …

Мощность в цепи

P = UI = I²R = U²/R
P_ист = ℰI   (полная мощность источника)

Правила Кирхгофа (напоминание)

Сумма токов в узле = 0 (со знаками). Сумма падений напряжений по замкнутому контуру = сумма ЭДС в контуре.
