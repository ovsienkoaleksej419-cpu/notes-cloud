⚛️ МКТ — молекулярно-кинетическая теория

Физика · Основные формулы, законы и графики

📘 Основные положения МКТ

  • Все вещества состоят из молекул
  • Молекулы находятся в непрерывном хаотическом движении
  • Между молекулами действуют силы притяжения и отталкивания
  • Столкновения молекул упругие

🔢 Основные формулы

Количество вещества:
N = ν · Nₐ
m = ν · M

Средняя кинетическая энергия:
Eₖ = 3/2 · kT

Уравнение состояния:
pV = νRT = NkT

Внутренняя энергия:
U = 3/2 · νRT

📊 Газовые законы

Изотермический процесс (T = const)

p · V = const

Изохорный процесс (V = const)

p / T = const

Изобарный процесс (p = const)

V / T = const

📈 Графики газовых процессов

p–V (T = const)

p–T (V = const)

V–T (p = const)
